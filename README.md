## 🐈‍⬛ Github Convention

### 이슈 제목 ➡️ [작업 종류] 이슈 이름 
ex) [Feat] 로그인 기능 추가 

### 브랜치 이름 ➡️  작업종류/#이슈 번호/적업할내용
ex) Feat/#3-Login

### 커밋 이름 ➡️ 작업종류: 이슈번호 - 작업내용  
ex) Feat:#3-로그인 버튼 추가
